Precomputes the spacetime metric components across the simulation grid. 

This function populates the global geometry array ({\ttfamily geom}) with the covariant metric, contravariant metric, and the metric determinant.

\begin{DoxyNote}{Note}
This function assumes {\bfseries{axisymmetry}}, where the geometry does not change with the azimuthal angle $\phi$ (represented by index $k$), therefore geom is a 2D array.

This function converts the raw energy and photon counts accumulated in the spectral grid into physical units. It calculates the SED across different inclination angles, determines the average optical depths, and computes global simulation diagnostics such as total luminosity and accretion efficiency.

{\footnotesize\ttfamily \+\_\+\+\_\+host\+\_\+\+\_\+ void set\+\_\+units (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{decs_8h_adf599310dc2d73d7dea6fe81566f2f1c}{Params}}}]{params }\end{DoxyParamCaption})}



Initializes and synchronizes physical units between the Host and the Device. 

This function defines the transformation scales between \char`\"{}code units\char`\"{} (dimensionless values used in the GRMHD simulation) and \char`\"{}\+CGS units\char`\"{} (physical values). The scaling parameters are defined as follows\+:


\begin{DoxyItemize}
\item {\bfseries{Mass Scaling}} ( $ M_{\rm unit} $)\+: User-\/defined mass unit to scale density and other mass-\/related quantities.
\item {\bfseries{Black Hole Mass}} ( $ M_{\rm BH} $)\+: Mass of the central black hole, used to define length and time scales.
\item {\bfseries{Length scale}} ( $ L_{\rm unit} $)\+: Defined as the gravitational radius $ R_g = GM_{BH}/c^2 $.
\item {\bfseries{Time scale}} ( $ T_{\rm unit} $)\+: Defined as the light-\/crossing time $ R_g/c $.
\item {\bfseries{Density scale}} ( $ \rho_{\rm unit} $)\+: Calculated as $ M_{\rm unit}/L_{\rm unit}^3 $.
\item {\bfseries{Magnetic Field scale}} ( $ B_{\rm unit} $)\+: Scaled such that the magnetic pressure $ B^2/4\pi $ matches the energy density scale $ \rho c^2 $ in Gaussian units.
\item {\bfseries{Energy Density Unit}} ( $ U_{\rm unit} $)\+: Defined as $ \rho_{ \rm unit} c^2 $.
\item {\bfseries{Number Density scale}} ( $ N_{e,\rm unit} $)\+: Assumes a fully ionized hydrogen plasma ( $ m_p + m_e $).
\end{DoxyItemize}

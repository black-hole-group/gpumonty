Constructs an orthonormal tetrad (local laboratory frame) for a moving observer. 

A tetrad represents a local Minkowski frame \$(e\+\_\+\{(0)\}, e\+\_\+\{(1)\}, e\+\_\+\{(2)\}, e\+\_\+\{(3)\})\$ at a point in curved spacetime.

\begin{DoxyNote}{Note}
This function uses the {\bfseries{Gram-\/\+Schmidt process}} to orthogonalize a set of coordinate basis vectors against the fluid\textquotesingle{}s 4-\/velocity.

Initializes simulation parameters with default values and parses command-\/line arguments. 

This function sets up the initial environment for the Monte Carlo simulation.

{\bfseries{Default Parameter Values\+:}}
\begin{DoxyItemize}
\item {\bfseries{seed}}\+: If not specified, the random seed is set to -\/1, which triggers time-\/based seeding for randomness.
\item {\bfseries{bias\+Tuning}}\+: Default bias tuning factor is set to 1.\+0. BIAS TUNING STILL NOT IMPLEMENTED.
\item {\bfseries{fit\+Bias}}\+: Default fitting bias factor is set to 1.\+0. FITTING BIAS STILL NOT IMPLEMENTED.
\item {\bfseries{fit\+Bias\+Ns}}\+: Scale input number of superphotons generated in each zone. Default is 1.\+0 (no scaling). FITTING BIAS STILL NOT IMPLEMENTED.
\item {\bfseries{tp\+\_\+over\+\_\+te}}\+: Sets the default proton-\/to-\/electron temperature ratio $ T_p/T_e $ for the simulation. Default is 3.\+0.
\item {\bfseries{beta\+\_\+crit}}\+: Sets the default critical plasma beta $ \beta_{\rm crit} $ for R-\/low/\+R-\/high models. Default is 1.\+0.
\item {\bfseries{trat\+\_\+small}}\+: Sets the minimum proton-\/to-\/electron temperature ratio $ T_p/T_e $ for R-\/low/\+R-\/high models. Default is 1.\+0.
\item {\bfseries{trat\+\_\+large}}\+: Sets the maximum proton-\/to-\/electron temperature ratio $ T_p/T_e $ for R-\/low/\+R-\/high models. Default is 10.\+0.
\item {\bfseries{Thetae\+\_\+max}}\+: Sets the maximum electron temperature $ \Theta_e $ for R-\/low/\+R-\/high models. Default is $ 10^{100} $.
\item {\bfseries{Black Hole Mass}}\+: Defaults to $ 4.1 \times 10^6 M_\odot $ (Sgr A$\ast$).
\end{DoxyItemize}

\begin{DoxyNote}{Note}
R-\/high and R-\/low models are only applied to iharm\+\_\+model simulations.
\end{DoxyNote}

GPUmonty is a high-\/performance, \href{https://docs.nvidia.com/cuda/cuda-c-programming-guide/}{\texttt{ CUDA}}-\/accelerated Monte Carlo radiative transfer (MCRT) code designed for the spectral modeling of accreting black holes based on \href{https://iopscience.iop.org/article/10.1088/0067-0049/184/2/387}{\texttt{ igrmonty}}.\hypertarget{md_README_autotoc_md1}{}\doxysection{1. Prerequisites}\label{md_README_autotoc_md1}
Before compiling, ensure your system has the following libraries installed and accessible\+:


\begin{DoxyItemize}
\item {\bfseries{\href{https://docs.nvidia.com/cuda/cuda-installation-guide-linux/}{\texttt{ CUDA Toolkit}}\+:}} Required for the {\ttfamily nvcc} compiler and GPU kernels.
\item {\bfseries{\href{https://www.gnu.org/software/gsl/}{\texttt{ GNU scientific library (GSL)}}\+:}} Required for various mathematical and statistical routines.
\item {\bfseries{\href{https://www.hdfgroup.org/solutions/hdf5/}{\texttt{ Hierarchical Data Format v5 (HDF5)}}\+:}} Required for reading GRMHD simulation snapshots.
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysubsection{Environment Configuration}\label{md_README_autotoc_md2}
Locate the installation paths for these libraries on your system and update the corresponding variables in the {\ttfamily Makefile}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{CUDA\_PATH     = /usr/local/cuda}
\DoxyCodeLine{GSL\_PATH      = /usr/local}
\DoxyCodeLine{HDF5\_INCLUDE  = /usr/include/hdf5/serial}
\DoxyCodeLine{HDF5\_LIB      = /usr/lib/x86\_64-\/linux-\/gnu/hdf5/serial}

\end{DoxyCode}
 After you have changed all these things, compile with by typing {\ttfamily make -\/j 15}. In case you want to compile it for debug, compile by typing {\ttfamily make BUILD\+\_\+\+TYPE=debug}.\hypertarget{md_README_autotoc_md3}{}\doxysubsection{Multi-\/\+Core Acceleration (\+Open\+MP)}\label{md_README_autotoc_md3}

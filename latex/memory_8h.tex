Allocates device memory for photon data using a Structure of Arrays (SoA) layout. 

Initializes a {\ttfamily \mbox{\hyperlink{structof__photonSOA}{of\+\_\+photon\+SOA}}} structure by allocating separate memory buffers on the GPU for every photon property for both original and scattered photons at each batch.
\begin{DoxyItemize}
\item Using a Structure of Arrays (SoA) instead of an Array of Structures (AoS) is a critical optimization that enables {\bfseries{coalesced memory access}} patterns, significantly increasing throughput for CUDA kernels.
\end{DoxyItemize}

Configures the optimal number of blocks to be used by CUDA kernels and generates GPU diagnostics. 

It multiplies the maximum number of blocks per multiprocessor by the total number of streaming multiprocessors (SMs) to determine the total maximum number of blocks that can be launched concurrently on the GPU.

◆ FIND_GCON_MATRIX_INV

#define FIND_GCON_MATRIX_INV   (0)

Macro to enable or disable the computation of the inverse metric using LU decomposition.

if set to 1, the code will compute the inverse metric through LU decomposition and substitution. if set to 0, the code will use the kerr analytical expressions for the inverse metric.
